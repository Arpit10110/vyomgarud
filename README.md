# VyomGarud - Military Grade UAVs

VyomGarud is a cutting-edge web application showcasing next-generation UAV systems designed for defense, surveillance, and critical missions. The platform highlights the precision, autonomy, and security of VyomGarud's drone technology.

## üöÄ Features

- **Modern & Responsive Design**: Built with a mobile-first approach, ensuring a seamless experience across all devices.
- **Glassmorphism UI**: Features a sleek, dark-themed user interface with glassmorphism effects for a premium look.
- **Interactive Animations**: smooth scroll animations using AOS (Animate On Scroll) and custom CSS animations.
- **Product Showcase**: Detailed cards displaying various drone models and their capabilities.
- **Contact Form**: A fully functional, responsive contact form for inquiries.
- **Dynamic Hero Section**: Engaging hero section with floating 3D-style imagery and gradient typography.

## üì¶ Getting Started

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/vyomgarud.git
    cd vyomgarud
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**

    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to [http://localhost:3000](http://localhost:3000) to view the application.
